# 论文格式智能排版工具

一个纯前端静态 Web MVP，用于展示在线论文排版工具的核心流程。

## 功能

- 类 Word 富文本编辑
- 左侧论文目录联动
- 右侧 A4 分页预览
- 论文信息与模板规则配置
- 格式检查
- 自动保存
- 项目 JSON 导入导出
- PDF 打印导出
- Word 文件导出

## 部署

这是纯静态站点，直接部署本目录即可：

- GitHub Pages：将本目录内容推送到仓库根目录，并开启 Pages。
- Vercel：导入仓库，Framework Preset 选择 Other，Output Directory 留空。
- Netlify：拖拽本目录或导入仓库，Publish directory 设为根目录。
